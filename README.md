## Descripción

Este proyecto automatiza la generación de documentos de Microsoft Word utilizando una plantilla y un archivo de Excel como fuente de datos.

El programa lee cada fila del archivo Excel, reemplaza automáticamente los campos correspondientes dentro de la plantilla de Word y genera un documento personalizado para cada registro. De esta manera, se agiliza la creación masiva de documentos, se reducen los errores y se optimiza el tiempo de trabajo.

---

## Características

- Lectura automática de datos desde Excel.
- Reemplazo de variables en una plantilla de Word.
- Generación de un documento por cada registro.
- Automatización del proceso.
- Reducción de errores y ahorro de tiempo.
